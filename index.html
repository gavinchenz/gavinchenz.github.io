# 坑我来踩，干货全拿走

【 😎 自述】

  • 本来呢就是因为好玩，自己也想弄个免费的博客玩玩，但又不知道内容上写点啥，于是，就将自己工作中遇到的，积累的，觉得好的东东，积攒在了一起，作为日常踩坑的工具文档了，有时间的时候就写写，没时间了就停下来先忙工作。因此自用嘛，也就没弄那么多花里胡哨的东西，具体内容在右上角菜单栏，有需要的自己去找哈。

  • 本人就一前端，文章主要以前端相关内容为主，都是日常踩坑，记录的一些东西，经手的项目有 ERP、CRM、GIS 等，pc 端主要技术栈是 VUE2 + Element-UI + RuoYi(大部分公司好像都这样吧，哈哈，不解释，dddd)；移动端的话就是 微信小程序 和 h5 的开发之前项目上接触过，鸿蒙 HarmonyOS 开发的话看过视频，因为项目中没有遇到，所以暂时没有过多涉及。

  • gis 相关的内容目前还在整理（因为太多了），工作项目中主要用到的就是 SuperMap GIS，因此对 iDesktop、iServer、iManager、iPortal 都很熟悉，对于空间数据处理、倾斜摄影、影像、地形、数据等服务的处理和发布也烂熟于心，前端 GIS 开发的话，也用的是 SuperMap 他们家的 iClient for Leaflet 和 iClient3D for Cesium；目前本文档内的 GIS 相关内容不太多，算是开个篇吧（我不是因为太多整理不过来哈），主要介绍下 SuperMap 家的 GIS 文档怎么用怎么看，另外就是作为一个前端开发，我们怎么开始自学入门，还有就是考虑到因为操作的内容比较多，因此纯文档的东西就不够看了，所以后期会考虑以文档 + 视频的方式记录（哎...为这又自学了个视频剪辑 😢 ），这个视频剪辑确实费时间，所以不保证更新速度哈（看心情更，哈哈 😎 ），但是呢，好奇心还是得有嘛，所以提供个最近才做的 尝鲜视频 (opens new window)看看得了，行不？ 😋

  • 后台开发的话，主要是跟着b站的黑马程序员的视频自学的，纯粹是为了记录下自己的学习进度。初衷一是因为自己干前端开发也有大几年的时间了，想拓展下自己的技术栈面，另一方面也是考虑因为工作中要经常为了项目开发和他人进行前后端沟通，为了提高沟通效率，想着学习下没啥坏处（我不会告诉你我是为了少踩坑才学的），毕竟到时候，后台懂得我也懂（后台），后台不懂的我还懂（前端），想想也挺 xb 的，对吧，哈哈

  • 说的确实有点多了，大家自己看吧，有用就拿去，没用就看看得了，留个邮箱 gzhlbs@163.com 有技术上的问题可以发邮件沟通交流，但拒绝催更哈，催更的一律回复：“忙” 😎
