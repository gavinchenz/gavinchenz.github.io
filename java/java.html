# 什么是 cmd ?

就是在windows操作系统中，利用命令行的方式去操作计算机。

我们可以利用cmd命令去操作计算机，比如：打开文件，打开文件夹，创建文件夹等。

打开 cmd： win + R 输入 cmd

打开 我的电脑： win + E

# 常见cmd 命令

  • 盘符名称 + 冒号 ---- 用于盘符切换

eg： E: 回车，表示切换到 E 盘

  • dir ---- 查看当前路径下的内容（包含被隐藏的文件及文件夹）

  • cd 目录 ---- 进入到单极目录

eg：cd SuperMap 指进入到当前目录下的 SuperMap 文件夹下

  • cd .. ---- 回到上一级目录
  • cd 目录1\目录2\ ... ---- 进入到多级目录下

eg：cd SuperMap\iServer\bin-

  • cd \ ---- 回到当前所属盘符根目录

  • cls ---- 清屏

  • exit ---- 退出cmd

# 类型转换

数字进行运算时，数据类型不一致不能运算，需要转成一样的，才能运算。

类型转换的类型：

  • 隐式类型转换（自动类型提示）

  • 强制类型转换

# 隐式类型转换

把一个取值范围小的数值，转成取值范围大的数值

--------------------------------------取值范围从小变大----------------------------------------->

byte --> short --> int --> long --> float --> double


// 举例：系统将隐式的（自动的）将整数 10 转换成 10.0 
int a = 10;
double b = a;
System.out.println(b); // 10.0

隐式转换的两种提升规则

  • 取值范围小的 和 取值范围大的 进行运算，小的会先提升为大的，再进行运算
  • byte short char 这三种类型的数据，在运算的时候，都会直接先提升为 int 类型，然后再参与计算

# 强制类型转换

如果把一个 取值范围大 的数值，赋值给 取值范围小 的变量，是不允许直接赋值的。必须先进行强制类型转换。

格式 ：目标数据类型 变量名 = (目标数据类型)被强制转换的数据


// 范例：
double a = 12.3;
int b = (int)a;
System.out.println(b); // 12

注意，强制转换时，如果超出强转类型的取值范围，会出现错误！！！


// 下面的强转会报错
int a = 300;
byte b = byte(a); // 会出现错误！！！

# 字符串的拼接

在一个+运算符的表达式中，存在字符串时，+就不再作为算术的“加”法，而是作为字符串的连接符使用。

"123" + 123 = 123123 // 字符串拼接

如果存在连续的 + 操作时，执行规则为从左到右逐个执行。


1 + 99 + "岁" // "100岁" 而不是 "199岁"

"我" + 1 + 8 + "岁" // "我18岁" 而不是 "我9岁"

# 字符的+操作

当 字符 + 字符 或者 字符 + 数字 时，会把字符通过 ASCII 码表查询到对应的数字再进行计算。


char a = 'a';
char b = 'b';
System.out.println(a + 10); // 97 + 10 = 108
System.out.println(a + b); // 97 + 98 = 195

// 当存在字符串时，+ 仅用于拼接字符串
System.out.println('a' + "abc"); // "aabc"

# 自增自减运算符

运算符 符号 作用 说明
自增 ++ 加 变量的值加 1
自减 -- 减 变量的值减 1

自增/自减运算符，即可以放在变量的前面，也可以放在变量的后面

当单独使用时，自增/自减运算符无论是放在变量前还是变量后，其结果都是一样的。


int a = 10;
int b = 10;
a++;
++b;
System.out.println(a); // 11
System.out.println(b); // 11
System.out.println(a == b); // true

int c = 10;
c++;
System.out.println(c); // 11
++c;
System.out.println(c); // 12
c--;
System.out.println(c); // 11
--c;
System.out.println(c); // 10

当参与计算时，自增/自减运算符放在变量前还是变量后，其结果遵循以下规律：

  • 放在变量后 ---- 遵循 先用后加 的原则
  • 放在变量前 ---- 遵循 先加后用 的原则

// 放在变量后 --- 先用后加
int a = 10;
int b = a++;
System.out.println(b); // 10
System.out.println(a); // 11

// 放在变量前 --- 先加后用
int c = 10;
int d = ++c;
System.out.println(c); // 11
System.out.println(d); // 11
System.out.println(c == d); // true

// 综合练习
int x = 10;
int y = x++;
int z = ++x;
System.out.println(x); // 12
System.out.println(y); // 10
System.out.println(z); // 12

# 关系运算符

# 短路逻辑运算符

符号 作用 说明
|| 短路或 结果和|相同，但因逻辑短路会减少逻辑判断从而减少性能损耗
&& 短路与 结果和&相同，但因逻辑短路会减少逻辑判断从而减少性能损耗

// 短路逻辑运算符具有短路效果
// 即，当左边的表达式能够确定最终的结果时，右边的就不再参与运行了
int a = 10;
int b = 10;
boolean result = ++a < 5 && ++b < 5;

System.out.println(result); // false
System.out.println(a); // 11
// 因为 a < 5 为 假， 已经足够确定 result 的结果，因此 && 右侧的 ++b 就不再运行了
System.out.println(b); // 10

# switch 语句
switch(表达式) {
    case 值1:
        语句体1;
        break;
    case 值2:
        语句体2;
        break;
    ...
    default:
        语句体n + 1;
        break;
}

switch 语句格式说明:

  • 表达式（将要匹配的值）：取值为 byte、short、int、char。JDK5 以后可以是枚举，JDK7 以后可以是 String。
  • case：后面跟的是要和表达式进行比较的值(被匹配的值)。
  • break：表示中断，结束的意思，用来结束 switch 语句。
  • default：表示所有情况都不匹配的时候，就执行该处的内容和 if 语句的 else 相似。
  • case 后面的值只能是字面量，不能是变量。
  • case 给出的值不允许重复

注意事项：

  1. default 可以省略但不建议省略，default 语句可以写在任何位置，但一般建议是写在末尾。
  2. case穿透：当 case 穿透开始后，后续的 case 语句就不会有匹配作用，内部的语句都会执行。直到看见 break或者 switch 语句执行完毕后才会结束。
  3. JDK12的新特性：switch 语句的简写模式

// JDK12的新特性: 箭头写法
int number = 1;
switch (number) {
    case 1 -> {
        System.out.println("一");
    }
    case 2 -> {
        System.out.println("二");
    }
    case 1 -> {
        System.out.println("三");
    }
    default -> {
        System.out.println("没有这个选项");
    }
}

// 如果语句体只有一句，`{}`大括号可以省略

int number1 = 2;
switch (number1) {
    case 1 -> System.out.println("一");
    case 2 -> System.out.println("二");
    case 1 -> System.out.println("三");
    default -> System.out.println("没有这个选项");
}

switch 语句 和 if 语句的第三种格式，其各自的使用场景：

  • if 语句的第三种格式：一般用于对范围的判断
  • switch 语句：把有限个数据一一列举出来，让我们任选其一

import java.util.Scanner;

public class Test8 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        System.out.println("请输入星期数");

        // 2. 判断
        /* // case 穿透
        switch (week) {
            case 1:
            case 2:
            case 3:
            case 4:
            case 5:
                System.out.println("今天是工作日");
                break;
            case 6:
            case 7:
                System.out.println("今天是休息日");
                break;
            default:
                System.out.println("输入的星期数有误");
                break;
        } */

        /* // 再次简化
        switch (week) {
